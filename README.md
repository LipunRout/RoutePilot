# 🧭 RoutePilot

> Smart route planning, simplified.

**RoutePilot** is a web application that helps users plan, visualize, and navigate routes with ease. Built with a modern JavaScript stack and deployed on Vercel.

🔗 **Live Demo**: [route-pilot.vercel.app](https://route-pilot.vercel.app)

---

## ✨ Features

- 🗺️ **Interactive Route Planning** — Plan routes between multiple destinations
- 📄 **PDF Export** — Download your planned route as a PDF for offline use
- ⚡ **Fast & Responsive** — Optimized for both desktop and mobile
- 🔄 **Real-time Updates** — Smooth loading states and instant feedback

## 🚀 Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/LipunRout/RoutePilot.git

# Navigate into the project
cd RoutePilot

# Install frontend dependencies
cd frontend
npm install

# Install backend dependencies
cd ../backend
npm install
```

### Running Locally

```bash
# Start the backend server
cd backend
npm start

# In a new terminal, start the frontend
cd frontend
npm start
```
